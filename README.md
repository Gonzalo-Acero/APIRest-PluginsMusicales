# API REST - PLUGINS MUSICALES

## Descripcion del proyecto
En este proyecto desarrolle una API Rest con Java 17 + Spring Boot 3 que consiste en un CRUD de plugins para trabajar el sonido.

## Teconologias usadas

* Java 17.
* Spring Boot 3
* Postman
* Git/Github Desktop
